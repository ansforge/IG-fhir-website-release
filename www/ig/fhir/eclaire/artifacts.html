Comportement: Capability Statements

La ressource CapabilityStatement permet de définir les capacités spécifiques que les systèmes doivent supporter pour être conforme à cet Implementation Guide (IG). Les systèmes conformes à cet IG doivent déclarer leur conformance à au moins une de ces ressources CapabilityStatement.

eclaire-consommateur

Système ECLAIRE (Essais CLiniques Accessibles Interconnectés pour la Recherche ouverts à l’Ecosystème)

Structures: Resource Profiles

Ces structures définissent les contraintes sur les ressources FHIR par les systèmes conformes à cet Implementation Guide (IG).

ECLAIRELocation

Profil de Location pour le projet ECLAIRE
ECLAIREResearchStudy

Profil de ResearchStudy pour le projet ECLAIRE

Structures: Extension Definitions

Ces structures définissent les contraintes sur les extensions FHIR par les systèmes conformes à cet Implementation Guide (IG).

ECLAIREConditionDetails

Extension créée dans le cadre du projet API ECLAIRE afin de permettre l’ajout de précisions sur le sujet concerné par l’essai
ECLAIREContactType

Extension créée dans le cadre du projet ECLAIRE qui indique le type de Contact : Public ou Scientific
ECLAIRELabel

Extension créée dans le cadre du projet API ECLAIRE afin de permettre l’ajout de plusieurs titres pour l’essai
ECLAIRERecruitmentPeriod

Extension créée dans le cadre du projet API ECLAIRE afin de permettre l’ajout de la période prévisionnelle de recrutement l’essai
ECLAIREReviewDate

Extension créée dans le cadre du projet API ECLAIRE afin de permettre l’ajout d’une date de révision fonctionnelle lorque que l’on souhaite indiquer la date de la dernière modification substancielle
ECLAIRESecondarySponsor

Extension créée dans le cadre du projet API ECLAIRE afin de pouvoir renseigner le(s) promoteur(s) secondaire(s)
ECLAIRESiteContactName

Extension créée dans le cadre du projet ECLAIRE pour indiquer le nom de contact spécifique au site de recrutement
ECLAIRETherapeuticArea

Extension créée dans le cadre du projet API ECLAIRE afin de permettre l’ajout du domaine thérapeutique concerné par l’essai
